position-areaを使用

ヘッダーのユーザメニュー

右上のアバターアイコンをクリックすると、ドロップダウンメニューが表示されます。
position-area: block-end でアイコンの下に自動配置しています。

gps_fixed

position-area

シンプルな位置指定, anchor()関数を使わずに直観的に配置できます。

sync

自動追従

ウィンドウリサイズ時も自動的に位置を追従します。

keyboard

アクセシビリティ

ESCキーで閉じる、背景クリックで閉じるなどの操作が自動実装されます。

コード例

/* アンカーの定義 * .user-icon { anchor-name: --user-icon; } /* ポップオーバーの位置 */ .dropdown-menu { position-anchor: --user-icon; position-area: block-end; }
